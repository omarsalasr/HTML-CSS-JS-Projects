If Thens

An if then is a tool that evaluates a logic statement. The basic premise is that it checks to see if the statement is true, and then uses the specified code.

In this example, the if then statement is checking that the variable y is equal to 1. if y is equal to 1 then program will use the system out statement. if, however, y is not equal to y, the program will not use the code below, and will continue to the next line.

You can also add a statement called an else, that allows you to chose between two different blocks of code depending on the result of the logic check.

If y is equal to 1, then the program will output “yay”, if it is not equal to 1, it will output “nay”.

The last tool you can use is called an else if, which essentially allows you to pair two if thens together

The same as always, if y is equal to 1 the program will output yay. If y is equal to 2 it will output maybe, and if it is neither 1 nor 2, it will output nay. One important thing to remember is that if both your if and your else if are true, only the code for the if will be used.
