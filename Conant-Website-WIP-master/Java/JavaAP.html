AP Java Review

About the test

The Advanced Placement Computer Science A test is the fastest growing of all AP courses. The test includes the development and analysis of algorithms and fundamental data structures, and the use of logic and formal methods. Out of ~50,000 students who took the test in 2016, 20.7% scored a 5 and 64.3% scored a 3 or higher.

Test Format

  • Multiple Choice Exam
    • There are 40 Multiple Choice questions
    • Total of 1 hour and 30 minutes given to complete
  • Free Response Question Exam
    • There are 4 Free Response Questions questions
    • Each question has multiple parts to them
    • Total of 1 hour and 30 minutes given to complete

Tips and Tricks

  • General Tips
    • Get a good night's sleep before the exam
    • Complete old AP test for extra practice
    • The longer questions usually are towards the end
    • You can write all over the test booklet
    • Quick reference page is not very useful when you are running out of time
    • Read and re-read all the note packets and annotate them until you fully understand key concepts
    • Do not be afraid to ask for help before the AP test
  • Multiple Choice
    • Bring multiple pencils for multiple choice
    • Make sure you have dull tip pencils, not mechanical or sharp
    • Know when to guess
    • If you can eliminate one or two of the choices, the odds are in your favor
    • Don't waste time on a problem that you are stuck on, move on to the next one
    • Check the output and look for the necessary lines that get you to that output
    • Read all of the answers before making your final decision
    • Trace the lines of code given, only if you have time
    • Check for loops that never execute or infinite loops
    • Check for nested if statements, sometimes they never execute
    • Be careful of instance variables vs local variables
    • You must know Demorgan’s law, it will be on the test
    • Recursion questions are simple, they are the easy point
    • When there are no braces around a condition or loop, the next line will be the only one that will execute as part of the condition
  • Free Response Questions
    • You don’t need to memorize any specific pieces of code, the AP test uses basic Java libraries
    • You can use any writing utencil to complete this portion of the test
    • There is a lot of partial credit
    • Make sure you have answered what the problem asks
      • Underline key words
      • You WILL NOT receive full credit if you don’t use the methods from a previous part of the same problem
      • Reread the problem definition after you think you've finished writing the code, paying close attention to the details you marked
      • Pre-conditions are the requirements needed for the method to work correctly, take them in full consideration
      • Post-conditions are the end result, the output, of the method
      • If you have time, trace your code to make sure you get the desired output
    • Aim for clarity in your programming
      • Organize, indent, assign meaningful variable names
      • Make sure to put the "private" keyword in front of all instance fields
      • Write neatly so the graders have an easier time reading your code
    • If you know how to do a small portion of the problem, write that code down
    • There is partial credit for having correct loops with no error
    • Write the correct header for any class or method to get easy points
    • No credit is given for writing a description of what you would do if you had time
    • Don't waste time erasing, draw a line through the lines you don't want to be graded
    • Never copy the headers that are written for you already
    • Don't panic if you run out of room to write
      • Find a blank page and continue your code
      • Indicate clearly what you've done, both on the page where you started the response and on the page where you continued writing
    • Avoid using classes that aren't part of the standard Java library
    • Don't forget the properties of interfaces
    • Remember the requirements for a super class and any sub class
    • Do not destroy or modify data was given, make a copy and modify if needed
